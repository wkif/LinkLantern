## 🎯 使用方式

### 搜索功能
1. 访问首页 `http://localhost:3000`
2. 点击选择搜索引擎（Google/Bing/百度/GitHub）
3. 输入搜索关键词
4. 点击"搜索"按钮或按回车键
5. 在新标签页打开搜索结果

### 查看链接
1. 登录账户
2. 首页自动显示您的所有链接
3. 点击分类按钮筛选特定分类的链接
4. 点击链接卡片访问网站（自动记录点击次数）

### 管理链接
- 点击"管理所有链接"按钮跳转到 `/links` 页面
- 或点击顶部导航的"我的链接"按钮

## 💡 技术实现

### useLinks Composable

**核心功能：**
```typescript
// 获取链接列表
fetchLinks(params?: { category?: string; search?: string })

// 创建链接
createLink(data: {...})

// 更新链接
updateLink(id: number, data: Partial<Link>)

// 删除链接
deleteLink(id: number)

// 记录点击
recordClick(id: number)
```

**响应式数据：**
- `links` - 链接列表
- `loading` - 加载状态
- `error` - 错误信息
- `categories` - 分类列表（自动计算）

## 🔄 数据流程

### 页面加载
1. 检查用户登录状态
2. 如果已登录，自动调用 `fetchLinks()`
3. 显示加载状态
4. 加载完成后展示链接卡片

### 点击链接
1. 调用 `recordClick(link.id)` 记录点击
2. 使用 `window.open()` 在新标签页打开链接
3. 不刷新当前页面

### 分类筛选
1. 自动从链接中提取所有唯一分类
2. 点击分类按钮过滤链接
3. 使用计算属性实时筛选
