# 🚀 浏览器书签导入 - 快速开始

## 5 分钟快速上手

### 步骤 1: 从浏览器导出书签 (2 分钟)

#### Chrome / Edge
```
1. 打开浏览器
2. 按 Ctrl+Shift+O (Windows) 或 Cmd+Shift+O (Mac)
3. 点击右上角的 ⋮ 菜单
4. 选择"导出书签"
5. 保存 HTML 文件到桌面
```

#### Firefox
```
1. 按 Ctrl+Shift+O (Windows) 或 Cmd+Shift+O (Mac)
2. 点击"导入和备份"按钮
3. 选择"导出书签为 HTML"
4. 保存文件到桌面
```

### 步骤 2: 登录 LinkLantern (1 分钟)

```
1. 打开 http://localhost:3000
2. 点击右下角"登录"按钮
3. 输入邮箱和密码
4. 登录成功后进入首页
```

### 步骤 3: 导入书签 (2 分钟)

```
1. 点击右侧用户头像
2. 选择"链接管理"
3. 点击"从浏览器导入"按钮
4. 选择"追加模式"（推荐）
5. 点击上传区域，选择刚才导出的 HTML 文件
6. 等待导入完成（约 5-30 秒）
7. 查看导入结果统计
```

### 完成！🎉

现在您可以：
- ✅ 在首页查看所有导入的链接
- ✅ 按分类筛选和查看
- ✅ 使用搜索功能快速查找
- ✅ 编辑和管理导入的链接

## 💡 小技巧

### 技巧 1: 分批导入
如果书签很多（1000+），建议分批导入：
```
1. 在浏览器中创建临时文件夹
2. 移动 300-500 个书签到临时文件夹
3. 导出并导入
4. 重复步骤 2-3
```

### 技巧 2: 选择性导入
只想导入某些分类的书签：
```
1. 在浏览器中创建"待导入"文件夹
2. 复制要导入的书签到该文件夹
3. 只导出"待导入"文件夹
4. 导入到 LinkLantern
5. 完成后删除临时文件夹
```

### 技巧 3: 保持同步
定期从浏览器导入新增书签：
```
1. 在浏览器中创建"新书签"文件夹
2. 新收藏的书签都放在这里
3. 每周导出该文件夹
4. 使用追加模式导入
5. 导入后在浏览器中清空该文件夹
```

## 🔍 故障排除

### 问题 1: 导入没有反应
```
原因: 文件格式不对或文件损坏
解决: 
1. 确认文件是 .html 或 .htm 格式
2. 重新从浏览器导出
3. 检查文件大小不为 0
```

### 问题 2: 部分书签没有导入
```
原因: 链接格式不正确或不是 http/https
解决:
1. 查看导入统计中的跳过数量
2. 检查原书签文件
3. 手动添加特殊格式的书签
```

### 问题 3: 导入很慢
```
原因: 书签数量太多
解决:
1. 耐心等待（1000 个书签约需 1-2 分钟）
2. 分批导入（每批 300-500 个）
3. 检查网络连接
```

### 问题 4: 图标显示不正常
```
原因: 图标获取失败或格式不支持
解决:
1. 导入完成后
2. 在链接管理页面
3. 点击"自动获取图标"按钮
4. 系统会重新获取图标
```

## 🎓 进阶使用

### 自动化导入脚本

可以使用 API 实现自动化导入：

```javascript
// 读取书签文件
const fs = require('fs')
const bookmarksHtml = fs.readFileSync('bookmarks.html', 'utf-8')

// 调用导入 API
fetch('http://localhost:3000/api/links/import-bookmarks', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer YOUR_TOKEN'
  },
  body: JSON.stringify({
    content: bookmarksHtml,
    mode: 'append'
  })
})
.then(res => res.json())
.then(data => console.log('导入结果:', data))
```

### 定时同步

结合浏览器扩展和定时任务，实现自动同步：

```javascript
// 伪代码
setInterval(() => {
  // 1. 从浏览器获取书签
  const bookmarks = chrome.bookmarks.getTree()
  
  // 2. 转换为 HTML 格式
  const html = convertToNetscapeFormat(bookmarks)
  
  // 3. 调用导入 API
  importToLinkLantern(html)
}, 24 * 60 * 60 * 1000) // 每天一次
```
